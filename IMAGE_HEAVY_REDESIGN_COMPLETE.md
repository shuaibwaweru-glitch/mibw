## Design System - Image-Heavy Components

### Image Gallery Grid
- 3-column responsive grid
- Images with overlay on hover
- Title, description, and link in overlay
- Smooth animations and scale effects

### Image Cards
- Square image at top
- Content below
- Hover effects: scale, shadow, lift

### Testimonial Avatars
- 80px circular images
- Gold border
- Centered layout
- Quote styling with serif font

### Team/Expert Grid
- Professional headshots
- 4-column grid responsive
- Name, title, bio, links
- Hover effects

### Before/After Section
- 2-column layout
- Large images
- Labeled sections
- Hover zoom effects

### Image Sections (Hero variants)
- Full background with gradient overlay
- Content overlay with text
- CTA buttons
- Background-attachment: fixed parallax effect
